CrowdWise needs an API key
This pilot app uses the Anthropic API to generate crowd intelligence for any city. You need a free API key to activate it — it stays in your browser only.
Get your key in 2 minutes:
1. Go to console.anthropic.com
2. Sign up free → go to API Keys
3. Click Create Key → copy it → paste below
Your key is saved locally in your browser and never sent anywhere except directly to Anthropic's API. Privacy policy
